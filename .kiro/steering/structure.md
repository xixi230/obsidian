# 项目结构

```
.
├── .obsidian/          # Obsidian 配置和插件
│   ├── plugins/        # 已安装插件
│   └── themes/         # 主题文件
├── .kiro/              # Kiro AI 助手配置
│   └── steering/       # 引导规则
├── 讲座/               # 主要教学内容
│   ├── lecture0/       # 第0讲：基础知识
│   │   └── 附件/       # 图片资源
│   ├── 工具/           # 开发工具教程
│   │   └── 附件/
│   ├── 语言基础/       # 编程语言入门
│   ├── 数学基础/       # 数学知识
│   └── 项目/           # 实践项目
│       └── 附件/
├── lecture0/           # 独立讲义（可能是草稿）
└── 论文/               # 参考论文
```

## 目录命名规范
- 使用中文命名目录和文件
- 图片资源统一放在对应目录下的 `附件/` 子目录
- 图片命名格式: `Pasted image YYYYMMDDHHMMSS.png`

## 内容组织
- 每个主题一个 Markdown 文件
- 相关图片与文档放在同一目录层级下的附件文件夹
