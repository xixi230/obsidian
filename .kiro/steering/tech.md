# 技术栈

## 文档系统
- **格式**: Markdown (.md)
- **编辑器**: Obsidian
- **版本控制**: Git (通过 obsidian-git 插件)

## Obsidian 配置
- 主题: GitHub Theme, Iridium
- 插件: obsidian-git, obsidian-style-settings, share-as-zip, yuque-publish

## Markdown 语法约定
- 使用 `![[filename]]` 语法嵌入图片（Obsidian 内部链接格式）
- 使用 `==文本==` 进行高亮标记
- 代码块使用标准 Markdown 围栏语法

# 无特定构建命令，纯文档项目
# 图片资源存放在各讲座目录下的 "附件" 文件夹

## 注意事项
- 图片引用使用 Obsidian 的 wikilink 格式，非标准 Markdown
- 文档主要使用中文编写
